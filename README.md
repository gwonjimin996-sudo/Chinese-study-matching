# 🇨🇳 중문과 중국어 회화 스터디 매칭 웹사이트

> 중문과 학생들이 중국어 회화 스터디를 모집하고 참여할 수 있는 웹 서비스

---

## 📖 프로젝트 소개

중국어 회화 실력 향상을 위해서는 꾸준한 대화 연습이 필요하지만 적절한 스터디원을 찾기 어려운 경우가 많습니다.

본 프로젝트는 중문과 학생들이 쉽게 스터디를 모집하고 참여할 수 있도록 지원하는 웹사이트입니다.

LocalStorage를 활용하여 별도의 서버 없이 동작하도록 구현하였습니다.

---

## ✨ 주요 기능

### 회원가입

* 이름 입력
* 학번 입력
* 이메일 입력
* 비밀번호 설정
* 사용자 정보 저장

### 로그인

* 학번 및 비밀번호 인증
* 로그인 상태 유지
* 로그아웃

### 스터디 모집

* 제목 작성
* 모집 인원 설정
* 유형 선택

  * 회화
  * HSK
  * 중국문화
* 활동 시간 입력

### 스터디 검색

* 키워드 검색
* 유형별 필터

### 스터디 신청

* 자기소개 작성
* 신청 내역 저장
* 중복 신청 방지

### 마이페이지

* 회원 정보 확인
* 내가 작성한 스터디 확인
* 신청한 스터디 확인

---

## 🛠 사용 방법

### 1. 회원가입

회원가입 메뉴에서 정보를 입력하여 계정을 생성합니다.

### 2. 로그인

학번과 비밀번호를 입력하여 로그인합니다.

### 3. 스터디 모집

스터디 모집 메뉴에서 모집글을 작성합니다.

### 4. 스터디 검색

홈 화면에서 키워드 검색 또는 유형 필터를 사용할 수 있습니다.

### 5. 스터디 신청

원하는 스터디 카드에서 신청 버튼을 눌러 자기소개를 작성 후 신청합니다.

### 6. 마이페이지

작성한 스터디 및 신청 내역을 확인할 수 있습니다.

---

## 📷 사용 예시

### 회원가입

입력

* 이름 : 김지민
* 학번 : 20250001
* 이메일 : [test@test.com](mailto:test@test.com)

결과

* 회원가입 성공

### 스터디 모집

입력

* 제목 : HSK 5급 스터디
* 모집인원 : 5명
* 유형 : HSK
* 활동시간 : 매주 수요일 18시

결과

* 스터디 목록에 등록

### 스터디 신청

입력

* 자기소개 : HSK 5급 준비 중입니다.

결과

* 신청 내역 저장
